This dashboard is part of a research project exploring cybersecurity in astrophysics data pipelines. It ingests real telemetry and space weather data, applies anomaly detection and displays results for educational and research purposes.

Data Sources

  • Parker Solar Probe magnetic field and plasma data (NASA SPDF)
  • Deep Space Network real‑time feed (NASA DSN Now)
  • NOAA space weather indices (NOAA SWPC)

Source code and documentation are available in the accompanying repository.
